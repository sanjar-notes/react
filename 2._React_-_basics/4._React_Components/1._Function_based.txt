====== 1. Function based ======
Created Wednesday 09 September 2020

* Used for dumb components (no state).
* Just return JSX.
{{{code: lang="js" linenumbers="True"
import React as 'react';

function App ()
{
	return (
			<div>
				<h1>Function-Based Component</h1>
			</div>
	);
}

export default App;
}}}
